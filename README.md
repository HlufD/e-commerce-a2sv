# Table of Contents

## Prerequisites

- Make sure you have the following installed on your machine:
  - Node.js(v20.x recommended)
  - npm or pnpm
  - Docker
  - PostgreSQL

## Installation

    - Clone the repository: git clone git@github.com:HlufD/e-commerce-a2sv.git
    - pmpm install

## Running the Project

    - Create a .env file in the root directory: cp .env.example .env
